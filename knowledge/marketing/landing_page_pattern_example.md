## High-Converting SaaS Landing Page Structure

### 1. Hero Section
- Clear, benefit-focused headline (max 10 words)
- Supporting subheadline explaining the "how"
- Primary CTA button (contrasting color)
- Hero image or product screenshot
- Trust badges (security, integrations)

### 2. Social Proof Bar
- Customer logos (5-7 recognizable brands)
- User count or metric ("Trusted by 10,000+ teams")
- Rating from G2/Capterra

### 3. Problem/Solution Section
- Identify 3 key pain points
- Show how product solves each
- Use icons for visual scanning

### 4. Features Grid
- 6-9 features maximum
- Icon + heading + short description
- Link to detailed feature pages

### 5. Pricing Preview
- Show 3 tiers: Starter, Pro, Enterprise
- Highlight most popular tier
- Annual discount prominently displayed
- "Free trial" or "No credit card required"

### 6. Testimonials
- 3 customer quotes with photos
- Include name, title, company
- Video testimonial if available

### 7. FAQ Section
- 5-7 common questions
- Accordion style for scannability
- Link to full help center

### 8. Final CTA
- Repeat primary value proposition
- Large CTA button
- Secondary link ("Talk to sales")
