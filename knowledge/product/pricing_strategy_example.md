## SaaS Pricing Tiers

### Tier 1: Starter (Free/$0)
**Target:** Individual users, hobbyists
- 1 user
- Basic features only
- 100 API calls/month
- Community support
- Watermark on exports
- **Purpose:** Lead generation, viral growth

### Tier 2: Pro ($29/month, $24/month annual)
**Target:** Small teams, professionals
- Up to 5 users
- All features
- 10,000 API calls/month
- Email support (24hr response)
- No watermarks
- Priority processing
- **Purpose:** Primary revenue driver

### Tier 3: Team ($79/month, $66/month annual)
**Target:** Growing teams
- Up to 20 users
- Everything in Pro
- 50,000 API calls/month
- Slack support
- Custom branding
- Analytics dashboard
- **Purpose:** Upsell from Pro

### Tier 4: Enterprise (Custom)
**Target:** Large organizations
- Unlimited users
- Unlimited API calls
- Dedicated account manager
- SLA guarantee (99.9%)
- Custom integrations
- On-premise option
- SSO/SAML
- **Purpose:** High-value accounts

## Pricing Psychology Applied
- **Anchoring:** Enterprise listed first or last for context
- **Decoy:** Team tier makes Pro look affordable
- **Annual discount:** 17% savings to improve LTV
- **Free tier:** Reduces friction, enables viral growth
